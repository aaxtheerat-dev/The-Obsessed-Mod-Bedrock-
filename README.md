# The Obsessed - Bedrock Horror Add-on

## Overview
"The Obsessed" is a horror-themed Minecraft Bedrock add-on inspired by the original Java mod. It adds a mysterious stalking entity that reacts to players, implements a sanity system, and includes animations, sounds, and camera effects to enhance the horror experience.

---

## Installation (Private Use)

1. **Import the Resource Pack**
   - Double-click `The_Obsessed_RP.mcpack` to import into Minecraft.
2. **Import the Behavior Pack**
   - Double-click `The_Obsessed_BP.mcpack` to import into Minecraft.
3. **Enable in a World**
   - Open world settings.
   - Enable **Experimental Gameplay**.
   - Enable **Holiday Creator Features**.
   - Enable **Beta APIs** (if required).

---

## Usage

- **Summon the Obsessed Entity**
- The entity will automatically stalk players, reduce sanity, and trigger camera/shaking effects.  
- Animations include:
- Idle breathing
- Slow walking
- Random twitch for horror effect
- Sounds:
- Breathing (`obsessed.breathing`)
- Whispering (`obsessed.whisper`)

---

## Features

- **Sanity System:** Player sanity decreases when near or being watched by the Obsessed.
- **Stalking AI:** The Obsessed teleports or vanishes when observed, creating a suspenseful experience.
- **Camera Effects:** Low sanity triggers screen shaking to increase tension.
- **Animations:** Fully animated entity for realistic movements.

## Notes

- Placeholders for textures and sounds are included:
- `textures/entity/obsessed.png`
- `sounds/breathing.ogg`
- `sounds/whisper.ogg`
